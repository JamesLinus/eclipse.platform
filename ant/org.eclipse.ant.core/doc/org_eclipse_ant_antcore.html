Ant Build Tool Core Plug-in

This document lists all of the extension points that the Ant Build Tool Core plug-in makes available to tool developers.

General Extension Points

The following extension points can be used to extend the fundamental capabilities of the Ant core plug-in:
  • org.eclipse.ant.antcore.objects
  • org.eclipse.ant.antcore.tasks
  • org.eclipse.ant.antcore.types
